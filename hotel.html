Descriptions

In this project, combining the lessons we were taught in CPSC 362: Software Engineering, my group aimed created a hotel booking system. This software will allow hotel employees to record and manage reservations, check-ins, and check-outs. This program will also generate booking/billing receipts for managers and send a confirmation email to the customer.


Technologies

We used Firebase, HTML, CSS, JavaScript


Challenges

It was the first project that required me to do a lot of independtly researching how to do accomplish certain tasks, such as how to create a user interface with HTML, CSS, and JavaScript. Also, I had to learn how to connect backend to frontend. Most of my previous projects have always been directly related to my class lessons and course work, but the lessons for this class only helped with how to plan and apporach our project. Since this was my first time learning HTML, CSS, and JavaScript, I was only able to gain a general understanding wih some of the basic features, so I wasn't able to accomplish the more complex features from my UX design. Also, I wasn't able to make my interface responsive to other screen sizes.


Outcomes

Successlly, the key features of the hotel booking system. I've learned the general concpets of frontend, but mostly importantly I learned from my mistakes of how to better structure the HTML and I need to incorporate repsonsiveness into the design.
